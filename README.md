# StockPro

منصة تحليل أسواق مالية احترافية مبنية باستخدام Streamlit و Python.

## المميزات

*   **تحليل فني متقدم:** مؤشرات RSI, MACD, Bollinger Bands, Ichimoku, VWAP وغيرها.
*   **ماسح ضوئي (Scanner):** البحث عن الفرص بناءً على معايير محددة (Screener).
*   **إدارة المخاطر:** حاسبة مخاطر مدمجة ومحفظة افتراضية.
*   **واجهة مستخدم عربية:** تصميم مخصص يدعم اللغة العربية (RTL).

## التثبيت والتشغيل

1.  تثبيت المتطلبات:
    ```bash
    pip install -r requirements.txt
    ```

2.  تشغيل التطبيق:
    ```bash
    streamlit run app.py
    ```

## التقنيات المستخدمة

*   Streamlit
*   Pandas
*   Plotly
*   Requests (Direct Yahoo Finance API)
